# Аудиорезак

Десктопное приложение для ноутбука: загрузить аудио, на волне выставить **два** фрагмента и сохранить их как **два отдельных файла** с выбранными именами.

## Стек

| Слой | Технология |
|------|------------|
| Язык | Python 3.10+ |
| Интерфейс | Tkinter (стандартная библиотека) |
| Волна | Pillow (сглаженная отрисовка, HiDPI) |
| Воспроизведение | pygame |
| Нарезка и декод | FFmpeg (`ffmpeg`, `ffprobe`) |
| Drag-and-drop (Windows) | [windnd](https://pypi.org/project/windnd/) |
| Сборка exe | PyInstaller |

Внешние зависимости, которые **не** ставятся через pip: **FFmpeg** должен быть установлен и доступен в `PATH`.

## Возможности

- Два выреза за один раз, свои имена и формат (по расширению файла).
- Волна с **перетаскиваемыми рамками** (края — границы, середина — сдвиг).
- Прослушивание всего трека и **предпрослушивание каждого выреза**.
- Загрузка: с диска, **по HTTPS-ссылке**, из **Telegram Bot API** (токен + `file_id`).
- Перетаскивание файла в окно (Windows).
- Темы: **Тёмная** и **Дофамин**.

## Требования

- **Windows** (основная цель; drag-and-drop и сборка exe рассчитаны на Windows).
- **Python 3.10+** — для запуска из исходников.
- **FFmpeg** — полная сборка с `ffmpeg` и `ffprobe` в `PATH`.  
  Скачать: [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)

Проверка:

```powershell
ffmpeg -version
ffprobe -version
```

## Установка (из исходников)

1. Клонируйте или скачайте репозиторий и перейдите в папку проекта:

```powershell
cd d:\audioCuter
```

2. (Рекомендуется) создайте виртуальное окружение:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. Установите зависимости:

```powershell
pip install -r requirements.txt
```

4. Запустите приложение:

```powershell
python main.py
```

## Сборка в `.exe`

1. Установите зависимости для сборки (один раз):

```powershell
pip install -r requirements.txt
pip install -r requirements-build.txt
```

2. Соберите:

```powershell
.\build.bat
```

или вручную:

```powershell
pyinstaller --noconfirm audioCuter.spec
```

3. Готовый файл: **`dist\AudioCuter.exe`**

На другом компьютере для exe нужны только **ffmpeg** и **ffprobe** в `PATH`. Python ставить не нужно.

Если exe не запускается из‑за pygame, попробуйте полную сборку:

```powershell
pyinstaller --noconfirm audioCuter.spec --collect-all pygame
```

## Использование

### Загрузка аудио

| Способ | Действие |
|--------|----------|
| Файл | **Загрузить аудио…** |
| Ссылка | **По ссылке…** — прямая `https://` на файл |
| Telegram | **Из Telegram…** — токен бота и `file_id` (файл должен быть доступен этому боту) |
| Drag-and-drop | Перетащите файл в окно (Windows, нужен `windnd`) |

Поддерживаемые форматы: mp3, wav, flac, m4a, aac, ogg, opus, wma и др. (через FFmpeg).

### Волна и рамки

- **Левый / правый край** рамки — начало и конец выреза.
- **Середина** рамки — сдвиг всего фрагмента.
- **Двойной щелчок** по волне — перемотка воспроизведения.
- Поля «Начало» / «Конец» синхронизируются с рамками (и наоборот).

Время можно вводить так:

- секунды: `12.5`, `90`
- `мм:сс`: `1:30`
- `ч:мм:сс`: `0:01:05`

### Воспроизведение

| Элемент | Действие |
|---------|----------|
| **▶ / ⏸** | Пауза / продолжить |
| **▶ Вырез 1 / 2** | Прослушать только выбранный фрагмент |
| **Пробел** | Пауза / продолжить (не в полях ввода) |
| **← / →** | Сдвиг позиции на ±1 с |
| **F1 / F2** | Прослушать вырез 1 или 2 |

### Экспорт

1. Задайте границы и **имена файлов с расширением** (например `intro.mp3`, `chorus.wav`).
2. Нажмите **Сохранить оба выреза…** и выберите папку.

Формат и кодек задаёт FFmpeg по расширению выходного файла.

### Темы

Переключатель **Тёмная** / **Дофамин** вверху окна.

## Частые проблемы

**«Не найдены ffmpeg/ffprobe»**  
Установите FFmpeg и добавьте папку с `ffmpeg.exe` и `ffprobe.exe` в переменную окружения `PATH`. Перезапустите терминал и приложение.

**«Не удалось построить огибающую»**  
Проверьте, что в файле есть звуковая дорожка и используется полная сборка FFmpeg (не урезанная portable без нужных кодеков). В сообщении об ошибке может быть текст от ffmpeg.

**Drag-and-drop не работает**  
Установите `windnd`: `pip install windnd`. Работает только на Windows.

**Пробел не ставит на паузу**  
Курсор в поле ввода — сначала кликните по волне или кнопке, затем используйте горячие клавиши.

**Telegram: ошибка getFile**  
Токен и `file_id` должны относиться к **одному** боту; бот должен иметь доступ к файлу (например, пользователь отправил файл этому боту).
